# 📖 Catalog Of My Things <a name="about-project"></a>

**Catalog Of My Things** is a simple console app that will help you to keep a record of different types of things you own: books, music albums, movies, and games, It's implemented using Object Oriented Programming principles on Ruby

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need to:

- Install [Git](https://git-scm.com/)
- Install [Ruby](https://rubyinstaller.org/)

### Setup

Clone the repo into your local machine:

```bash
git clone https://github.com/BahirHakimy/Ruby-Capstone-Project.git
```

### Usage

To start the project execute this command

```bash
ruby main.rb
```

### Run tests

To run the tests execute this command

```bash
rspec spec/
```
